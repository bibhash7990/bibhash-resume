# Unode - Web3 RPC Node Platform

## Overview
Unode is a developer platform and node service console that provides Web3 RPC endpoints, wallet integrations, and decentralized protocol utilities for the Unification network.

## Main Features & Functionality
- **Node Endpoint Management**: Developers register and manage custom API keys and RPC endpoints for Ethereum/EVM and Unification protocols.
- **Payment Streams (Superfluid)**: Integrated real-time money streaming for subscriptions, enabling paying per node request incrementally.
- **Oracle Services**: Dashboards for configuring Verifiable Oracle Randomness (VOR) and Oracle of Oracles (OoO) contract feeds.
- **Testnet Faucet**: Web UI for requesting test tokens on the unification test network.
- **Visual Analytics**: Charting of node latency, request counts, and billing metrics.

## Tech Stack & Integrated Tools
- **Frontend**: React, React Router, Redux, Chart.js, SCSS, Bootstrap.
- **Blockchain & Web3**: Web3.js, Viem, MetaMask wallet integration.
- **Subgraphs & API**: Apollo GraphQL client for indexing, GraphQL queries.
- **Payment Gateway**: Superfluid Finance SDK (Streaming Widgets and Core).
