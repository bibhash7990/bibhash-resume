# LIFTREX - Client Portal (Elevator lifecycle)

## Overview
LIFTREX is a white-label client-facing portal that integrates with the Skyline elevator lifecycle system, allowing property owners, developers, and building managers to view project status, proposals, and schedules.

## Main Features & Functionality
- **Proposal Viewer**: Review and sign proposals for new elevator installations and service contracts.
- **Project Progress Dashboard**: Real-time Gantt/project tracking showing engineering, manufacturing, installation, and inspection stages.
- **Task Management**: Collaborating with contractors and engineers on pending punch lists.
- **Payments & Ledger**: Inspect outstanding balance, past invoices, and upcoming payment schedules.

## Tech Stack & Integrated Tools
- **Frontend**: React, TypeScript, React Query, Radix UI, Tailwind CSS, Axios.
- **Backend & Database**: Express, Node.js, MongoDB (shared with Skyline core database).
- **Core Integrations**: PDF-generation utilities, document download helper, responsive mobile layout.
