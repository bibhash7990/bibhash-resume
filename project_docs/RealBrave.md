# RealBrave - EdTech e-Learning Platform

## Overview
RealBrave is an interactive EdTech e-learning platform connecting students, parents, and teachers with real-time class sessions, scheduling, and analytics.

## Main Features & Functionality
- **Zoom Virtual Classrooms**: Real-time virtual learning rooms integrated via Zoom Video SDK.
- **Skill Assessments**: Dynamic evaluations, quizzes, and grading dashboards to assess student performance.
- **Teacher & Parent Portals**: Role-specific dashboards giving teachers grading capabilities and parents student progress metrics.
- **Real-Time Messaging**: WebSockets-driven chat between teachers, parents, and students.
- **Rich Dashboards**: Interactive charts mapping student grade trends and attendance metrics.

## Tech Stack & Integrated Tools
- **Frontend**: React, Vite, Redux Toolkit, React Router, Bootstrap, Sass.
- **Video & RTC**: Zoom Video SDK UI Toolkit, react-media-recorder, react-webcam.
- **Real-Time Sync**: Socket.io-client, Firebase.
- **Visual Analytics**: ApexCharts, React ApexCharts.
- **Scheduling**: React Big Calendar.
