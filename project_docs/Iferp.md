# IFERP - Academic Research & Event Portal

## Overview
IFERP is a massive scientific research community and academic conference management portal that connects students, professionals, institutions, and corporate entities.

## Main Features & Functionality
- **Global Research Profile**: Academic research dashboard where members display profiles, track publications, journals, and citations.
- **Conference & Event Booking**: Advanced registration for scientific conferences, summits, and symposiums.
- **Walkthrough Guide (Joyride)**: User-guided onboarding walkthroughs for new profiles and event setup.
- **Multi-Gateway Payment Integration**: CCAvenue, PayPal, and Razorpay support for international registrations.
- **Interactive Certificate Canvas**: Custom certificate rendering engine using Fabric.js to generate printable researcher awards.

## Tech Stack & Integrated Tools
- **Frontend**: React, Redux Toolkit, SCSS, Bootstrap, Bootstrap Icons.
- **Backend & Cloud**: AWS SDK (S3 File storage), Firebase, Socket.io-client.
- **Document Rendering**: Fabric.js, html2canvas, html2pdf.js, jspdf, pdf-lib.
- **User Analytics**: PostHog-js.
- **Payment Providers**: PayPal, Razorpay, CCAvenue.
- **Scheduling**: React Big Calendar.
