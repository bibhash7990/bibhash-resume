# SANAD-HUB - Invoice & Financial Management Platform

## Overview
SANAD-HUB is a bilingual (Arabic / English) invoice, quotation, and financial management SaaS platform featuring complete RTL layout capabilities.

## Main Features & Functionality
- **Invoice & Bill Modules**: Full-featured billing systems enabling users to create invoices, track quotes, log payments, and manage expenses.
- **Bilingual RTL Support**: Fully localizable layout adapting natively to Arabic language flows.
- **Customizable Invoicing Templates**: Drag-and-drop elements enabling customization of brand invoices.
- **Multicurrency & Tax Systems**: Configurations for local tax rates, multiple currencies, and automatic currency conversion ledger.
- **QR Code Scanning**: QR Code encoding on invoice previews for quick scanning and authentication.

## Tech Stack & Integrated Tools
- **Frontend**: React, React Router, Redux Toolkit, Styled Components, Joy UI, Material UI.
- **Drag & Drop**: DnD Kit, react-beautiful-dnd, react-draggable.
- **Data Export & Files**: xlsx, file-saver.
- **Integrations**: qrcode, crypto-js, moment, dayjs.
