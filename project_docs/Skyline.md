# Skyline - Elevator Project Lifecycle Management

## Overview
Skyline is a specialized operations platform designed to manage the entire lifecycle of elevator installation, maintenance, proposals, and inspections for elevator service companies and building inspectors.

## Main Features & Functionality
- **Proposal Dashboards**: Dynamic creation and tracking of hydraulic, traction, and MRL elevator installation proposals.
- **Inspection Workflows**: Automated scheduling and documentation of building safety inspections and regulatory compliance approvals.
- **Procurement Tracker**: Tracking procurement of elevator parts, mechanical components, and logistics.
- **Gmail-Linked Inbox & Communications**: Integration of Gmail API to sync user correspondence directly inside the platform.
- **Billing & Payments**: Integrated ledger mapping contracts, payment schedules, and invoices.

## Tech Stack & Integrated Tools
- **Frontend**: React 18, Vite, Redux Toolkit (redux-persist), TanStack React Query, Radix UI, Tailwind CSS, Axios.
- **Backend & Database**: Hapi.js 21, Node.js, MongoDB (Mongoose), hapi-swagger.
- **Integrations**: AWS S3 (documents upload), Google OAuth 2.0, Gmail API, Twilio (notifications), Nodemailer, Dropbox, Adobe PDF Services, wkhtmltopdf (invoice export), ExcelJS, node-cron.
